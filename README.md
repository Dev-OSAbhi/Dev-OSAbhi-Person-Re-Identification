## 📂 Dataset Setup

Follow the steps below to correctly set up the dataset for this project.

### 1️⃣ Create the Data Folder
Create a folder named **`data`** in the **root directory** of the project.

### 2️⃣ Download the Dataset
Download the dataset from the link below:

🔗 **[Download Dataset](https://zenodo.org/records/17909410)**

⚠️ **Important:**  
Make sure to download **`extracted.zip`** and **NOT** `original.zip`.

### 3️⃣ Extract the Dataset
After downloading:

- Extract **`extracted.zip`**
- Rename / keep the folder as **`DB_extracted`**
- Place it inside the **`data`** folder.
